# 🚀 Git Practice Project

Учебный проект для отработки навыков работы с Git, PyCharm и виртуальными окружениями Python.

## 🛠 Технологии
* **Python 3.14**
* **Git** (ветвление, слияние, алиасы)
* **uv** (быстрое управление окружением)
* **PyCharm** (IDE)

## Чему я научилась в этом проекте:
* Инициализации локального репозитория (`git init`).
* Добавлению и исправлению коммитов (git commit -m "", git ci --amend)
* Настройке алиасов для ускорения работы (`co`, `ci`, `st`, `hist`).
* Работе с ветками: создание, переключение и безопасное удаление.
* Слиянию веток (`merge`) с использованием стратегии Fast-forward.
* Управлению зависимостями через современный инструмент **uv**.
* Настройке `.gitignore` для чистоты репозитория.
* Связыванию локального кода с удаленным сервером по **SSH**

## Как запустить:
1. Убедитесь, что у вас установлен [uv](https://github.com).
2. Склонируйте репозиторий.
3. Создайте и активируйте окружение:
   ```bash
   uv venv
   .\.venv\Scripts\activate
4. Запустите основной файл:
