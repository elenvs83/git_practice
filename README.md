# Git Practice Project

Учебный проект для отработки навыков работы с Git, PyCharm и виртуальными окружениями Python.

## Чему я научилась в этом проекте:
* Инициализация локального репозитория (`git init`).
* Настройка алиасов для ускорения работы (`co`, `ci`, `st`, `hist`).
* Работа с ветками: создание, переключение и безопасное удаление.
* Слияние веток (`merge`) с использованием стратегии Fast-forward.
* Управление зависимостями через современный инструмент **uv**.
* Настройка `.gitignore` для чистоты репозитория.

## Как запустить:
1. Убедитесь, что у вас установлен [uv](https://github.com).
2. Склонируйте репозиторий.
3. Создайте и активируйте окружение:
   ```bash
   uv venv
   .\.venv\Scripts\activate
4. Запустите основной файл:
python main.py
